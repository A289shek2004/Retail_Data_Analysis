📊 Retail Data Analysis

A complete data analysis project focused on understanding sales performance and demand trends for PEXPO product lines using real-world retail sales data. This project demonstrates key skills in exploratory data analysis (EDA), data preprocessing, visualization, time-series forecasting, and actionable business insights.

🧠 Project Objective

To analyze retail sales data of products and generate insights that can help improve:

Product assortment decisions

Inventory planning

Regional demand forecasting

Strategic business growth

📁 Dataset

The dataset includes sales transactions with fields such as:

Order Date

SKU / Product

Sales Revenue

Units Sold

Region / City

Platform (Online / Offline)

Customer Segment

🔧 Tools & Technologies

Python: Pandas, NumPy, Matplotlib, Seaborn

Machine Learning: Scikit-learn (Regression, Time-Series)

SQL: Querying and data extraction skills

Business Analytics: KPI calculation, trend analysis

🛠 Key Tasks & Workflow
1️⃣ Data Cleaning & Preprocessing

Handled missing values and duplicates

Standardized date formats and categorical fields

Prepared clean datasets for analysis and modeling

2️⃣ Exploratory Data Analysis (EDA)

Analyzed sales trends across regions and time periods

Identified top-selling products and slow-moving SKUs

Visualized sales distributions and patterns

3️⃣ Modeling & Forecasting

Built regression and time-series models to forecast future sales

Evaluated models based on accuracy and generalization

Used models to provide demand estimates for planning

4️⃣ Business Insights

Provided recommendations on inventory allocation

Suggested focus areas based on seasonal demand data

📊 Key Insights

Sales Trends: Identified peak sales months and lag periods

Top Performers: SKU-level analysis to find high-revenue products

Regional Demand: City-level demand variation and clustering

Forecasting Results: Projected future sales for planning
